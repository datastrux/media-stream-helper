🔍 Troubleshooting

If media is not detected:
  1. Refresh this page after installing the extension
  2. Make sure you click "Play" on the media element
  3. Wait a few seconds after playback starts
  4. Check the browser console (F12) for any errors
  5. Verify the extension is enabled at chrome://extensions/

📋 Testing Checklist

After testing, verify these features work:

  • ✓ Media URLs appear in the extension popup
  • ✓ Correct media type badges are shown (MP4, MP3, HLS, etc.)
  • ✓ "Copy URL" button copies the full URL
  • ✓ "Copy FFmpeg" buttons generate valid commands
  • ✓ "Download File" button works for direct media (not HLS)
  • ✓ "Clear All" button removes detected media
  • ✓ Different tabs show different detected media
  • ✓ Timestamps are displayed correctly
